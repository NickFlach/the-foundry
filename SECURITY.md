# Security Policy

## Reporting Vulnerabilities

If you discover a security vulnerability, please report it to **nflach78@gmail.com**.

**Do not** open a public issue for security vulnerabilities.

## Response Time

- **Acknowledgment**: within 48 hours
- **Resolution**: as fast as we can

## Attribution

We don't have a bounty program yet, but we will credit you in the fix if you'd like.

We take security seriously. Thank you for helping keep The Foundry safe.
